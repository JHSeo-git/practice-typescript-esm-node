# practice-typescript-esm-node

Practice for typescript + native esm + node

## Folder structure

- packages
  - practice-library
    - code: node + native esm + typescript library
    - build: cjs + esm + d.ts
- apps
  - practice-node-cjs: compile to cjs
  - practice-node-esm: compile to esm
